# PokéMénage — Guilde familiale V4

Application familiale optimisée pour iPad et publiée avec GitHub Pages.

## Nouveautés V4

- Conseil de guilde hebdomadaire.
- Six métiers adultes et six métiers apprentis pour les enfants.
- Plus de 90 idées de tâches ménagères.
- Parcours des huit arènes de Kanto et présentoir des badges.
- Journal des tâches réalisées.
- Correction du membre ayant accompli une tâche avec transfert automatique des XP.
- Annulation d’une validation erronée.
- Sauvegarde automatique dans le navigateur et export/import JSON.

## Mise à jour GitHub Pages

1. Remplacez les fichiers de votre dépôt par le contenu de ce dossier.
2. Dans GitHub Desktop, créez un commit : `Mise à jour PokéMénage V4`.
3. Cliquez sur **Push origin**.
4. Patientez une à trois minutes, puis rechargez l’application sur l’iPad.

Les anciennes sauvegardes locales sont migrées automatiquement. Les tâches réalisées avant la V4 restent validées, mais seules les nouvelles validations apparaîtront dans le journal modifiable.
