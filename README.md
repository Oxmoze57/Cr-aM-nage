# PokéMénage — Guilde familiale V4.3

Application familiale optimisée pour iPad et publiée avec GitHub Pages.

## Nouveautés V4.3 — Souvenirs de guilde

- Suppression de la récompense liée à la capture.
- Les combats donnent maintenant des victoires d’arène, sans récompense matérielle ou virtuelle.
- Nouvel onglet **Souvenirs** avec album photo familial.
- Ajout d’une photo, d’un titre, d’une date, d’un membre, de son métier et d’une anecdote.
- Création automatique d’un souvenir lors de l’obtention d’un badge.
- Les photos souvenirs sont réduites avant stockage pour limiter leur poids.
- Les souvenirs et leurs photos sont inclus dans la sauvegarde JSON.

## Fonctionnalités principales

- Conseil de guilde hebdomadaire.
- Six métiers adultes et six métiers apprentis pour les enfants.
- Plus de 90 idées de tâches ménagères.
- Parcours des huit arènes de Kanto et présentoir des badges.
- Journal des tâches réalisées avec correction de la personne et transfert des XP.
- Photos de profil pour chaque membre.
- Sauvegarde automatique, export, restauration et réinitialisation.

## Mise à jour GitHub Pages

1. Faites d’abord une sauvegarde dans **Réglages**.
2. Remplacez les fichiers de votre dépôt par le contenu de ce dossier.
3. Dans GitHub Desktop, créez un commit : `Ajout des souvenirs de guilde V4.3`.
4. Cliquez sur **Push origin**.
5. Patientez une à trois minutes, puis fermez et rouvrez l’application sur l’iPad.

L’adresse GitHub Pages reste la même et les anciennes sauvegardes locales sont migrées automatiquement.
